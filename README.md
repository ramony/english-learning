# 英语学习助手

一个帮助儿童学习英语的 React 应用。

## 功能特点

### 1. 分级学习
- 小学一年级到初三的词库
- 每个年级对应不同难度
- 适合年龄的主题分类
- 循序渐进的学习内容

### 2. 词库内容
- 一年级：基础词汇（水果、动物、颜色、数字）
- 二年级：学校用品、家庭成员、天气
- 三年级：运动、食物、时间
- 四年级：爱好、日常活动
- 五年级：地理、自然、职业
- 六年级：科技、环境、文化
- 初中：更复杂的主题词汇

### 3. 学习功能
- 分类学习
- 单词发音
- 音标显示
- 选择题模式
- 错题本功能

### 4. 游戏化元素
- 连续答对计数
- 成就系统
- 每日学习目标
- 进度追踪

### 5. 数据统计
- 学习进度统计
- 正确率统计
- 每日目标完成度
- 错题记录

## 词库结构

每个年级包含多个主题分类，每个单词包含：
- 英文单词
- 中文含义
- 表情图标
- 音标标注
